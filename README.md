### 使用方法
第一次运行时，可以使用**Hsmod配置管理器**自动安装[HsMod](https://github.com/Pik-4/HsMod)插件（会自动从github拉取最新Release）。启动游戏正常后点击重连按钮，会连接至插件内置本地服务器进行管理。
